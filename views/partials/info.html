Parsing the PDF

Pass the link of the PDF attachment from Trello that you'll be working on.

Sample Link

Copy each record and paste it on the Details sidebar by hovering the mouse on either side of the PDF.

Ensure that each detail is correct. Editing when necessary.

Click Save.

Saving the extracted Data

Edit or remove rows from the data table by clicking on the Lock icon and double-clicking on the row you wish to edit.

Use the Filter icon to view all Invalid rows. (Data without a phone number or have an employee size below 10.)

Click the Download icon to download the data table as a CSV file.
